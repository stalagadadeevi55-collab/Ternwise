# Project Laddu Travel Planner

Installable SPA/PWA MVP for privacy-aware group travel planning.

## Current Stack

- React + Vite
- PWA manifest and service worker
- Supabase planned for auth, database, and row-level security
- Capacitor planned for Android and iOS packaging after the web MVP proves useful

## MVP Command

```powershell
npm install
npm run dev
```

## Product Notes

- Read [docs/mvp-spec.md](docs/mvp-spec.md) for the MVP scope captured from the product interview.
- Read [src/data/schema-notes.md](src/data/schema-notes.md) for the first Supabase data model direction.

## Store Path

- Web/PWA first for friends and family testing.
- Android Play Store later with Capacitor Android app bundle.
- iOS App Store later with Capacitor, Xcode, App Store Connect, and Apple Developer Program membership.
